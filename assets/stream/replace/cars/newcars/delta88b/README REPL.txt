Installation (replace) 

NOTICE: BACKUP BEFORE INSTALLING. OpenIV have a neat feature you can have mods cars and maps, while keeping the game file original, for more info go to  http://openiv.com/?p=1132



1. extract all the files into a directory.

2. Rename the yft, ytd files to match the car you want to replace

3. For best result, modify the vehicles, handling and carvariation files (data provided below) 

4. Defragment the rpf files.
